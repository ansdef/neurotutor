# NeuroTutor — Выживи в первые дни за рубежом

Платформа для выживания на английском языке. Реальные ситуации, реальные разговоры — без воды.

## Что это

Интерактивный тренажёр английского для тех, кто переезжает, учится или работает за границей. Не языковой курс — набор для выживания на первую неделю.

- **Персональный план на 7 дней** — составляется по целям, стране и ситуациям
- **Практика по сценариям** — паспортный контроль, регистрация в аэропорту, поиск жилья
- **Мгновенная обратная связь** — грамматика, тон, улучшенный вариант ответа
- **Два языка** — русский (по умолчанию) и английский, переключение в один клик

## Запуск

Откройте `index.html` в браузере — никаких зависимостей и сборки не нужно.

## Деплой

### GitHub Pages
1. Зайдите в репозиторий → **Settings → Pages**
2. Source: **Deploy from a branch**, ветка **main**, папка **/ (root)**
3. Сайт будет доступен по адресу `https://ansdef.github.io/neurotutor/`

### Netlify
1. Перетащите папку проекта на [netlify.com/drop](https://app.netlify.com/drop)
2. Готово — Netlify выдаст публичную ссылку

### Vercel
1. Зайдите на [vercel.com/new](https://vercel.com/new)
2. Импортируйте репозиторий `ansdef/neurotutor` из GitHub
3. Framework preset: **Other**, всё остальное — по умолчанию

## Ветки

| Ветка | Содержимое |
|-------|-----------|
| `main` | Статический сайт |
| `bot`  | n8n workflow для Telegram-бота |

## Telegram-бот

В ветке `bot` находится файл `Telegram_Language_Learning_Bot2.json` — n8n workflow с:
- Telegram-триггером (текст и голос)
- Транскрипцией через Groq Whisper
- AI-агентом на базе OpenRouter
- Клавиатурой с быстрыми ответами
